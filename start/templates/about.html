About

Oh sup, thanks for looking at the about page. We definitely didn't just throw this together yesterday. That's why everything is spelled correctly...

Here at queuesome we're all about giving people a way to listen to music together in a fun and interactive way.

You take turns choosing different vibes and everyone submits a song that they think fits the best.

You can throw some likes if you thought their choice was choice too... And that's it!

To play you just need one person with a Spotify Premium account to kick things off and the rest can join in.

If you want s'more info about other features and how it works, take a look below at the next section.


Thanks again for checking us out!



Getting Things Going


{% load static %}

Kick things off by starting your own sesh. Once you log in to a Spotify Premium account, you'll be prompted to choose a playback device...

A playback device is anything linked to your Spotify account. Could be a phone, laptop, smart speaker, etc...


{% load static %}

Enter a fun name for your sesh and give yourself a name people will know you by


{% load static %}

When you're all set, share your room code with everyone else so they can join.

And if they're too lazy to type the url in, hit the QR button and they can scan it and come right to the join page


{% load static %}

If you're looking to join someone else's sesh, type in the sesh's room code and give yourself a name


Let's Go

Once everyone is in, hit the Let's Go to get things going, and if anyone wants to join later, that's okay too!


Game Flow


{% load static %}

When the game starts, a person is randomly chosen to select the vibe for the round


{% load static %}

You can choose one of the vibes from our list or...


{% load static %}

...create your own vibe with the custom option


{% load static %}

When the vibe is submitted for the round, everyone get's to pick a song to match the current vibe


{% load static %}

You can type in the name of a song you want or... be more generic and type in an artist's name to search their popular songs


{% load static %}

Take a look through the results and pick the song you searched for... if it's not there or you want something else, hit back and try again


{% load static %}

When everyone submits their song choice, each song will play for 80 seconds (the host can always change this in Settings)

Throw the song a thumbs up if you like it or a thumbs down if you're not really feeling it


{% load static %}

When the round is over, check out the results page to see which songs were totally a vibe


and that's usually how it goes. Once everyone submits their picks for the round, someone else will get to choose the next vibe and the rest of the rounds will seamlessly flow together



Special Features


{% load static %}
queuesome twosome

If two or more people choose the same song for a vibe, that's called a queuesome twosome. This means that the whole song will play and nobody will get credit if other people like it


song skip

If everyone in the sesh isn't feeling it and they all thumbs down the song, then the song will stop and the next song will play



{% load static %}
Scattergories

Pick the Scattergories vibe to randomly generate a letter and everyone will have to pick a song that starts with that letter.


{% load static %}
By Artist

Get more specific with the vibe and enter in an artist for everyone to submit a song by


Settings


{% load static %}

If you created the sesh you can go to the settings page to change how long songs should play and what device to play from

If things die down or you just got something else better to do, hit the Kill Party button to close out the party. And if you forget, don't worry, we'll close it out for you after nothing has been done for a while


{% load static %}

You can check out the user page to see who's in the party and who may be holding up the line. If you started the sesh, you can remove users from the sesh if they left or you can skip ahead if they'll be back but didn't pick a song yet
